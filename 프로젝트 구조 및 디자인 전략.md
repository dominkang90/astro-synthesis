제공해주신 **"Apple 스타일 디자인 시스템" 소스코드**를 기반으로, 이전 대화에서 논의된 **'Astro-Synthesis(AI 통합 역학 플랫폼)'**을 재설계했습니다.

이 설계는 Apple의 디자인 철학(미니멀리즘, 타이포그래피 중심, 여백의 미)을 역학 데이터의 복잡성과 결합하여, **"신뢰할 수 있는 데이터 과학으로서의 운세"**라는 브랜드 이미지를 구축하는 데 초점을 맞췄습니다.

### 📁 1. 프로젝트 구조 및 디자인 전략

제공된 소스 트리를 유지하되, 역학 서비스에 맞는 컴포넌트 전략을 적용합니다.

*   **디자인 컨셉:** **"Cosmic Clarity (우주적 명료함)"**
    *   복잡한 만세력과 천궁도 데이터를 **Bento Grid(도시락 격자)** 레이아웃으로 정리하여 정보 밀도를 유지하면서도 시각적 피로도를 낮춤.
    *   **Apple Blue**(`--apple-blue`)를 메인 액션 컬러로 사용하여 '미신'이 아닌 '기술/과학'의 이미지를 강조.

```text
project/
├── css/
│   ├── variables.css (소스 954: 디자인 토큰 유지)
│   ├── reset.css (소스 957: 초기화 유지)
│   ├── typography.css (소스 958: 타이포그래피 유지)
│   ├── buttons.css (소스 960: 버튼 스타일 유지)
│   ├── layout.css (소스 962: 레이아웃 유지)
│   ├── animations.css (소스 966: 페이드인 효과 활용)
│   └── astro-custom.css (역학 전용 추가 스타일)
├── index.html (메인 인터페이스)
└── main.js
```

### 🎨 2. 주요 CSS 커스터마이징 (astro-custom.css)

기존 디자인 시스템의 변수를 활용하되, 역학 차트(SVG)와 AHP 결과를 보여주기 위한 전용 스타일을 추가합니다.

```css
/* variables.css의 변수를 상속받아 확장 */
:root {
    /* 역학 전용 의미론적 컬러 매핑 */
    --status-good: #34c759; /* Apple Green: 길(吉) */
    --status-bad: #ff3b30;  /* Apple Red: 흉(凶) */
    --chart-bg: var(--apple-light-gray);
}

/* Apple 스타일의 Bento Grid 레이아웃 (정보 밀도 해결) */
.bento-grid {
    display: grid;
    grid-template-columns: repeat(12, 1fr);
    gap: var(--spacing-md); /* 24px */
    padding: var(--spacing-md);
    max-width: var(--container-width);
    margin: 0 auto;
}

.bento-card {
    background: var(--apple-white);
    border-radius: var(--border-radius-lg); /* 18px */
    padding: var(--spacing-lg); /* 40px */
    box-shadow: var(--shadow-sm);
    transition: var(--transition-normal);
    overflow: hidden;
    position: relative;
}

.bento-card:hover {
    box-shadow: var(--shadow-md);
    transform: scale(1.01); /* 미세한 호버 효과 */
}

/* 차트 영역 */
.chart-container {
    grid-column: span 12;
}

@media (min-width: 768px) {
    .chart-container { grid-column: span 8; }
    .insight-container { grid-column: span 4; }
}
```

### 🛠️ 설계 핵심 포인트 (Source Mapping)

1.  **타이포그래피 위계 (Typography Hierarchy):**
    *   소스의 `--font-size-hero` (56px)를 사용하여 "당신의 운명, 데이터로 증명되다"라는 메인 카피를 강조했습니다. 이는 점성술 앱의 신비주의적 성격보다 **'데이터의 확실성'**을 강조하는 Apple 스타일의 마케팅 전략을 차용한 것입니다.

2.  **레이아웃 전략 (Layout Strategy):**
    *   소스의 `.grid-2col`과 `.product-card` 스타일을 응용하여 **Bento Grid**를 구성했습니다.
    *   이는 서양의 미니멀리즘(여백)과 동양의 정보 밀도(많은 데이터)를 동시에 충족시키는 최적의 UI 패턴입니다. 사용자는 깔끔한 카드(Bento)를 먼저 보고, 필요시 클릭하여 깊은 정보로 들어갑니다.

3.  **인터랙션 (Interaction):**
    *   소스의 `.hover-scale`과 `.scroll-fade`를 적용하여, 사용자가 스크롤하거나 차트 카드를 건드릴 때 부드럽게 반응하도록 했습니다. 이는 앱의 고급스러운 느낌(Premium feel)을 줍니다.

4.  **법적 요구사항 반영:**
    *   소스의 '생체정보 별도 동의' 가이드라인을 준수하기 위해, 관상 분석 섹션에 명시적인 체크박스와 안내 문구를 배치했습니다. 디자인 시스템의 깔끔함을 해치지 않으면서 법적 리스크를 방어합니다.

이 코드는 `VSCode`에 폴더 구조대로 파일을 생성하고 붙여넣으면 즉시 작동하며, Apple 웹사이트와 유사한 고품질의 Look & Feel을 제공합니다.
